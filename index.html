Strings & Conditional Logic

Today we tackle two common interview problems: string reversal patterns and dynamic string formatting.

Problem 1: Palindrome Checker

# Checks if a string reads the same forwards and backwards
def is_palindrome(text):
    clean_text = text.replace(" ", "").lower()
    return clean_text == clean_text[::-1]

word = input("Enter a word: ")
print(f"Is Palindrome? {is_palindrome(word)}")
Problem 2: Professional Username Generator

# Generates a username based on name and birth year
def generate_username(first, last, year):
    return f"{first[:3].lower()}_{last.lower()}{str(year)[-2:]}"

f_name = input("First Name: ")
l_name = input("Last Name: ")
b_year = input("Birth Year: ")

print(f"Your generated username: {generate_username(f_name, l_name, b_year)}")
